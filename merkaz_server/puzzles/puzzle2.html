Part 2

        You will receive multiple input lines.

        Each line containes digits from (1-9).

        For each line, you must construct the largest possible two-digit number under the following rule:

        Choose two digits from the line.

        The second digit must appear after the first in the line.

        The digits do not have to be adjacent.

        The order of digits cannot be changed.

        The two selected digits form a two-digit number (AB).

        After processing all lines, sum all the two-digit numbers.

        Return the final sum.

    
Example 1

    Input : 4736

    Possible two-digit numbers (valid order): 47, 43, 46, 73, 76, 36

    Greatest: 76

    
Example 2

    Input line: 9271

    Greatest valid number: 97

    
Output

    A single number - the sum of the greatest two-digit numbers from all lines.
